# Authentication-App
An Android App that allows a user to sign up/login and logout from the app using MVVM and CLEAN architecture.

## Technologies used:
* Kotlin
* Android Studio
* XML
* Themes
* Material Design (Google)
* Laravel (PHP & MySQL)
* Retrofit
* okHttp
* Kotlinx Coroutines
* MVVM architecture
* Dagger Hilt
* DataStore
* Navigation Architecture Components
